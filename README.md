# rules-us-ky

Kentucky rules in Akoma Ntoso XML format.

## Structure

- `statutes/` - Kentucky Revised Statutes (KRS) in Akoma Ntoso XML
- `regulations/` - Kentucky Administrative Regulations (KAR) in Akoma Ntoso XML

## Akoma Ntoso

[Akoma Ntoso](http://www.akomantoso.org/) is an OASIS standard XML schema for legislative and judiciary documents, enabling machine-readable legal text with semantic markup.

## Sources

- [Kentucky Legislature](https://legislature.ky.gov/) - Official source for Kentucky Revised Statutes
- [Kentucky Administrative Regulations](https://apps.legislature.ky.gov/law/kar/titles.htm) - Official Kentucky regulations
